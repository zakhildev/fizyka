# Wykresiki na fize

Wykresiki ładne na fizykę ;)

## How to uruchomić

### Instalacja

#### Linux / macOS

```bash
$ python -m venv . # Tworzysz tu virtualne środowisko
$ source ./bin/activate # Aktywujesz środowisko (Linux/macOS)
$ python -m pip upgrade pip # Update pip'a do najnowszej wersji
$ pip install -r requirements.txt
```

#### Windows

```powershell
$ python -m venv .
$ Set-ExecutionPolicy -ExecutionPolicy Undefined -Scope CurrentUser
$ ./bin/activate.ps1
$ python -m pip upgrade pip
$ pip install -r requirements.txt
```

Polecenie z plikiem `activate` musisz odpalać za każdym razem jak chcesz pracować ze środowiskiem. Jupyter i VSCode automatycznie powinny to wykrywać.
